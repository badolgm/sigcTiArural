# Recursos del Laboratorio de Matemáticas Avanzadas

Este directorio contiene recursos estáticos y archivos HTML para el laboratorio de matemáticas avanzadas.

## Estructura:
- `advanced_math_labv2.html` - Laboratorio principal de matemáticas
- `simulations/` - Simulaciones interactivas
- `assets/` - Recursos adicionales (imágenes, scripts, etc.)

## Integración:
Los archivos HTML se integran con el componente React `AdvancedMathLab.jsx` mediante iframes o enlaces directos.
